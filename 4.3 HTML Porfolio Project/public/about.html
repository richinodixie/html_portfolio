About Me

Hi there! My name is Richard, and I want to develop a CBT website for people suffering from anxiety!

I took four classes in Computer Science (incl. Web Dev) in undergrad, but didn't like them. But I find online self-help websites immensely helpful (ha) after using one myself. Mojo was able to help me with my psychogenic erectile dysfunction issues, so I wanted to build a website like it to help more people!
